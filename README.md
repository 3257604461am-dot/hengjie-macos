# 横截

当前版本：**0.11.0（构建 16）**

横截是一款原生 macOS 菜单栏工具，支持普通截图、上下与左右滚动长截图、可编辑截图草稿、钉住区域、提取文字、选区 GIF 录制和本地剪贴板历史。图片、OCR、翻译、GIF 编码和历史数据均在本机处理，不使用横截网络服务。

## 功能概览

- 普通截图与原位标注：画笔、箭头、形状、高亮、文字、序号、马赛克和水印。
- 最近截图自动保存原始底图与可编辑标注图层，并按普通、横向、纵向分类。
- 延时截图，以及自由、固定比例、自定义比例和固定像素选区。
- 上下、左右滚动长截图，支持手动与自动滚动；0.8.0 使用连续帧、多尺度边缘匹配和重复纹理安全检查。
- 钉住区域，将图片长期置顶并继续标注。
- 中文、日语、英语 OCR，macOS 15 及以上支持苹果本地翻译。
- 选区 GIF 录制，支持 1–30 FPS、三档质量和鼠标显示选项。
- 类似 Windows `Win+V` 的剪贴板历史，支持文字、富文本、链接和静态图片，并可搜索与筛选。
- 可从剪贴板图片历史直接打开原图继续标注。
- 自定义菜单栏功能面板、分类设置页、快捷键录制和登录时启动。
- 本地滚动诊断日志与问题诊断 ZIP 导出，不包含用户截图或剪贴板正文。
- 0.10.0 优化框选、标注、截图打开和历史加载流畅度，并降低长截图、剪贴板监听和日志写入的后台占用。
- 0.11.0 集中提升长截图可恢复性、历史存储可靠性、标注会话安全和发布诊断能力；不降低截图质量。

## 0.10.0 性能优化

- 普通截图捕获后立即显示编辑器，截图草稿的无损 PNG、缩略图和工程文件改为后台保存。
- 框选时缓存可吸附窗口并按屏幕刷新节奏合并绘制，避免鼠标移动时反复枚举全部系统窗口。
- 标注撤销只记录发生变化的图层，不再为每次拖动复制全部标注；画笔和马赛克轨迹会合并过密采样点。
- 长截图帧通过非主线程有界缓冲传递，匹配进度合并刷新，处理落后时不会无限堆积图像。
- 图片保存直接使用 ImageIO 编码 `CGImage`，不再经过 TIFF 中间格式；历史图片也在后台解码。
- 剪贴板监听在活跃时保持 250ms 响应，空闲后自动降至 750ms，并在系统睡眠或功能关闭时停止。
- 动画遵循 macOS“减少动态效果”设置；所有优化保持原始截图、草稿和导出结果无损。

## 0.11.0 可靠性与发布基础

- 长截图会记录不含像素的版本化检查点，并提供可注入的帧回放入口；匹配置信度不足时暂停，避免输出错误长图。
- 捕获会话使用唯一令牌隔离异步回调，旧会话不会覆盖新编辑器；标注工程使用稳定图层 ID 和差异式撤销记录。
- 剪贴板历史与最近截图索引迁移到 SQLite WAL，旧版 JSON 会保留迁移备份，载荷和标注工程仍独立存储。
- 工程拆分为 Core、Capture、Annotation、History、Media 和 App 六个边界；ScreenCaptureKit、标注模型、SQLite 及 OCR/GIF 编码不再全部堆在应用目标内。
- 默认发布包仍是轻量 Apple Silicon ad-hoc 包；正式构建脚本预留 Developer ID、Hardened Runtime、公证和 Sparkle 更新接口。

## 系统要求

- Apple Silicon Mac
- macOS 14 或更高版本
- 普通截图需要“屏幕录制”权限
- 滚动截图默认手动，不需要“辅助功能”权限；只有主动切换自动滚动时才需要该权限
- 提取文字在 macOS 14 可用；中英日系统翻译需要 macOS 15 或更高版本

## 安装与首次运行

1. 将 `横截.app` 移到 `/Applications`。
2. 首次打开时，根据提示授予“屏幕录制”权限；授权后需要退出并重新打开应用。
3. 如需自动滚动，在“系统设置 → 隐私与安全性 → 辅助功能”中允许横截。
4. 菜单栏出现取景框图标后，点击可打开横截功能面板；也可按 `⌥⇧A` 普通截图，按 `⌥⇧P` 钉住区域，按 `⌥⇧O` 提取文字，按 `⌥⇧G` 录制 GIF，按 `⌃⇧V` 查看剪贴板历史。

如果 macOS 阻止首次启动，可在 Finder 中右键点击横截，选择“打开”。这是个人本机使用的 ad-hoc 签名版本，未进行 Developer ID 公证。

如果系统设置中已经打开“横截”但应用仍显示未授权，通常是旧版本或另一运行路径留下的权限记录。请执行：

```sh
tccutil reset ScreenCapture com.wonderlab.hengjie
```

然后确认只从 `/Applications/横截.app` 启动，重新授权并完全退出、重新打开一次。0.2.1 起使用稳定的本地指定要求，后续覆盖升级不会再因每次构建产生不同的 ad-hoc `cdhash`。

## 使用方法

- 普通截图：菜单栏选择“普通截图”或按 `⌥⇧A`，拖出选区后直接在原选区标注；点击“完成”会把编辑结果复制到剪贴板并关闭截图界面。
- 延时截图：先框选区域，再选择 3、5 或 10 秒倒计时。倒计时期间可操作目标应用，右键、`Esc` 或控制条“取消”可终止。
- 框选顶部浮动栏可切换自由选区、`1:1`、`4:3`、`16:9`、自定义比例或固定像素尺寸。固定像素会按起点所在显示器的 Retina 比例换算，并同时显示逻辑尺寸与输出像素。
- 进入框选模式时屏幕保持原色；左键或触控板单点拖动进行框选，只有开始拖动后选区外才会变暗。右键、触控板双指点按或 `Esc` 可随时取消。
- 上下长截图：菜单栏选择“上下长截图”，框选滚动内容区域。
- 左右长截图：菜单栏选择“左右长截图”，框选横向滚动内容区域。
- 长截图默认手动滚动；如需自动模式，点击“手动滚动”切换，并将鼠标停留在截图区域。鼠标移出区域时自动滚动会暂停且不会移动鼠标指针。
- 0.8.0 起，单显示器长截图使用连续帧流，并自动从最近帧中寻找可靠的中间位置。采集中仍不要移动或缩放目标窗口。
- 匹配不可靠、检测到重复纹理、动画干扰、窗口变化或反向滚动时，横截会暂停，不会强行生成错误长图；可重试当前段、继续手动滚动或完成已拼接部分。
- 长图完成后会自动以“适合窗口”比例显示整张图片，可继续放大、缩小、恢复 100% 后检查和编辑；点击“复制”会复制原始像素尺寸的编辑结果。
- 普通截图和长图标注界面提供选择、画笔、直线、箭头、矩形、椭圆、高亮、文字、序号、马赛克、水印、撤销/重做、OCR、复制和保存；文字可双击修改，箭头端点和马赛克轨迹可再次调整。“钉住区域”不混入截图编辑流程。
- 最近截图：菜单栏选择“最近截图”。首次使用确认本地隐私说明后，后续普通截图、横向长图和纵向长图会自动建立草稿；标注修改约 500ms 后合并写入。普通截图完成、长图复制或保存后会标记为已完成。
- 最近截图最多保留 100 条、30 天且总容量不超过 2GB。可按类型筛选、重新编辑、复制、另存、删除或清空。关闭记录后不会新增或更新草稿，已有记录默认保留，也可选择关闭并清空。
- 钉住区域：菜单栏选择“钉住区域”或按 `⌥⇧P`，框选后图片会直接显示在最上层，不进入截图编辑器，也不会修改剪贴板。
- 钉图顶部工具栏提供画笔、高亮、马赛克、颜色、透明度、粗细、撤销、复制和“解除钉”。复制后钉图继续保留；解除钉只关闭图片，不修改剪贴板。
- 钉图可以拖动、等比例缩放并同时存在多张；它们会跨桌面空间和全屏应用显示，在横截退出时统一消失。
- 提取文字：菜单栏选择“提取文字”或按 `⌥⇧O`，框选后会打开可编辑的文字结果，不保存截图，也不会自动修改剪贴板。
- 文字识别支持简体中文、繁体中文、日语和英语。0.5.2 起直接识别屏幕位图，并采用兼顾速度和准确率的模式，不再执行 TIFF 编解码。
- macOS 15 及以上可在简体中文、日语和英语之间翻译。原文语言默认自动识别，也可通过下拉框手动纠正；翻译完成后可分别复制译文或双语对照。
- 翻译首次使用某个语言方向时，macOS 会请求下载苹果语言包；请允许下载并等待完成。失败时结果框会显示当前语言方向、具体原因和“重试翻译”。
- 录制 GIF：菜单栏选择“录制 GIF”或按 `⌥⇧G`，在单个显示器内框选固定区域。设置 `1–30 FPS`、高清 `100%` / 标准 `75%` / 小文件 `50%` 和是否显示鼠标后，点击“开始录制”。
- GIF 录制有 3 秒倒计时；录制时选区正上方会显示可拖动的便携控制栏，包含录制状态、时长、大小估算、FPS、质量、鼠标状态、停止和取消。点击或切换到其他应用时控制栏仍会保持置顶，也可再次按 `⌥⇧G` 停止。
- 停止后会循环预览，可选择保存、重新录制或放弃。取消会删除临时文件且不打开预览。便携控制栏和红色边框都不会进入成品。
- GIF 最长录制 5 分钟，不录制声音；超大选区会等比缩小到最长边 4096 像素且单帧不超过 1600 万像素。红色边框和控制条不会进入成品。
- 剪贴板历史：菜单栏选择“剪贴板历史”或按 `⌃⇧V`。首次使用需确认本地隐私说明，确认后只记录后续发生的剪贴板变化，不导入启用前的内容。
- 历史浮窗支持关键词搜索、全部/文字/链接/图片/已固定筛选、鼠标点击、方向键选择、回车复制、`Esc` 关闭、固定、删除和清空未固定记录。多个搜索词必须全部命中，匹配忽略大小写、重音和全角/半角差异。
- 历史支持纯文本、链接、RTF、HTML、PNG、JPEG 和 TIFF；不记录 GIF、Finder 文件、音视频以及带密码、临时或自动生成标记的内容。记录在本机保留 30 天、最多 100 条、总计不超过 1GB，固定记录不会自动删除。
- 图片历史会在后台生成缩略图，但不会自动 OCR；图片可按类型、时间和固定状态筛选。图片行的铅笔按钮或右键“编辑图片”会读取原始 PNG/JPEG/TIFF 并打开标注编辑器，不使用缩略图作为底图。
- 普通截图、钉图、提取文字、GIF 录制和剪贴板历史快捷键都可以在分类设置页中直接录入、清除或恢复默认。组合冲突或已被其他应用占用时不会覆盖当前有效配置。

## 问题诊断

- 从菜单栏面板或“设置 → 诊断”选择“导出问题诊断”，可生成 `横截-诊断-日期时间.zip`。
- 诊断包包含最近 7 天的横截运行日志、版本、macOS、芯片、显示器、权限状态和可读取的横截系统崩溃报告。
- 诊断日志与 Instruments signpost 只记录功能状态、阶段耗时、拼接分数、失败原因和资源错误，不记录截图、钉图、GIF、OCR/翻译文本、剪贴板正文或快捷键配置。
- 日志仅保存在本机，最多 20MB，不会自动上传。

## 本地构建

构建应用和命令行质量门禁只依赖 macOS Command Line Tools：

```sh
swift run HengJieCoreChecks
swift run HengJieArchitectureChecks
./scripts/build-app.sh
```

完整 XCTest 需要 Xcode，在本机或 GitHub Actions 中执行 `swift test`。生成的应用和压缩包位于 `outputs/`。

默认构建为 Apple Silicon ad-hoc 包。正式签名构建可设置 `SIGNING_MODE=notarized`、`DEVELOPER_ID_APPLICATION` 和 `NOTARY_PROFILE`；没有 Apple Developer 凭据时不会尝试公证。Sparkle 更新仅在正式签名构建中启用，ad-hoc 包保持轻量并关闭自动更新。

## 隐私与权限

完整说明见 [PRIVACY.md](PRIVACY.md)。

- 屏幕画面、可编辑截图草稿、OCR、翻译、GIF 和剪贴板历史均在本机处理，横截不运行服务器，也不上传内容。
- “屏幕录制”权限用于读取用户选中的屏幕区域。
- “辅助功能”权限只在用户主动开启自动滚动时使用。
- 苹果翻译语言包首次下载可能需要联网，翻译过程由 macOS 系统提供。
- 剪贴板历史保存在 `~/Library/Application Support/横截/ClipboardHistory/`。可在设置中停止监听或清空全部记录；第三方应用若未正确标记敏感数据，横截无法自动判断其是否为密码。
- 最近截图保存在 `~/Library/Application Support/横截/ScreenshotHistory/`，底图、缩略图和标注工程分开存储；截图历史与剪贴板历史使用独立开关。
- 诊断日志保存在 `~/Library/Application Support/横截/Diagnostics/`，保留 7 天且总量不超过 20MB；只有用户主动导出时才会生成诊断 ZIP。

## 已知限制

- DRM 视频、系统保护窗口、游戏独占画面和持续动画内容可能无法捕获或稳定拼接。
- 自动滚动依赖目标控件正确响应系统滚轮事件；特殊控件请切换为手动滚动。
- 横向与纵向拼接优先适配浏览器、Excel、WPS 和飞书表格；纯重复纹理、低纹理或持续动画仍可能触发安全暂停。
- 首版 GIF 只录制固定屏幕区域，不跟随窗口，不提供暂停、逐帧编辑、裁剪、标注、变速、音频或 MP4 导出。
- 当前公开 ZIP 为 Apple Silicon 个人 ad-hoc 版本；正式签名、公证和 Sparkle 更新接口已预留，但需要发布者自己的 Apple Developer 凭据。
- macOS 14 仅支持文字提取、编辑和复制，不支持系统翻译；翻译按钮会显示版本要求。
- 苹果系统语言包被拒绝下载、网络不可用或系统翻译服务异常时无法完成翻译；原文和已有译文会保留，可恢复网络后重试。
- 横截只能记录应用运行且历史功能已启用期间的剪贴板变化；极短时间内连续覆盖剪贴板时，只能保存监听时实际可见的最后一项。
